Gets the default ResourceReaper session id.

The default ResourceReaper will start either on GetAndStartDefaultAsync(IDockerEndpointAuthenticationConfiguration, CancellationToken) or if a ITestcontainersContainer is configured with ITestcontainersBuilder<TDockerContainer>.WithCleanUp.
