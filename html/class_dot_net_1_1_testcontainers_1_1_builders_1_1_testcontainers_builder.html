Detailed Description

This class represents the fluent Testcontainer builder. Each change creates a new instance of ITestcontainersBuilder<TDockerContainer>. With this behaviour we can reuse previous configured configurations and create similar Testcontainer with only little effort.

var builder = new TestcontainersBuilder<TestcontainersContainer>()
.WithName(&quot;nginx&quot;)
.WithImage(&quot;nginx&quot;)
.WithEntrypoint(&quot;...&quot;)
.WithCommand(&quot;...&quot;);
<br />
var http = builder
.WithPortBinding(80, 08)
.Build();
<br />
var https = builder
.WithPortBinding(443, 443)
.Build();
